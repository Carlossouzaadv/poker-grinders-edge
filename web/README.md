## 🚀 Como Usar

### Executar Localmente
```bash
npm install
npm run dev
```
Acesse: http://localhost:3002 (ou porta disponível)

### Testar o Replayer
1. Cole um hand history do PokerStars na textarea
2. Clique em "📊 Analisar Mão"
3. Use os controles para navegar pela hand
4. Teste o "🎯 Usar Exemplo" para demo rápida
